Bigfoot's Forest

Bigfoot's Forest est un jeu d'horreur façon jeu rétro codé sur Unity pour la Scream Jam 2022+1.

Vous y incarnez un journaliste tentant de prouver que Bigfoot n'existe pas en explorant une forêt locale.

Langage

C#

Unity

Type de Jeu

Jeu d'horreur

Temps de développement

1 semaine

Caractéristiques du jeu

- On peut se déplacer dans toutes les directions et interagir avec certains objets.

- Certains objets sont requis pour continuer d'avancer dans le jeu.

- Bigfoot se déplace aléatoirement dans les niveau, et attaque le joueur si il le croise.

- Si le joueur se fait attraper par Bigfoot, il doit recommencer le niveau.
