Site Maraîcher (SAE 1.04)

Le but de cette SAE fut de créer un CRUD pour une association de maraîchers afin de pouvoir ajouter, modifier, supprimer et voire différentes tables et entités.

Pour cela, nous avons dû créer un MCD regroupant toutes les tables requises par l'association, puis nous avons réalisé le CRUD ainsi qu'une page de statistiques.

Mon travail

Durant ce projet, j'ai :

- Aidé à la réalisation du MCD du site.

- Crée une partie du script SQL du site (maraîchers, produits).

- Crée le modèle MVC du site au niveau des modèles et des Contrôleurs.

- Crée le CRUD pour la table Vend du site.

- Crée le logo du site en prenant en compte le sujet.

- Crée une page de statistiques montrant les meilleurs maraîchers avec leur photo.



Les Compétences

J'ai développé les compétences suivantes pendant la SAE :

- Créer une base de données à partir d'un dialogue avec un client. (C4)

- Dialoguer avec un client. (C5)

- Créer une application Web. (C1)

- Créer une application en modèle MVC. (C1)

- Répartir le travail selon les compétences des membres du groupe. (C6)



Retour réflexif

Au final, je pense avoir bien réussi cette SAE. J'ai fait ce qu'il fallait que je fasse et le résultat m'a convenu dans l'ensemble.

J'aurais cependant aimé coder des CRUD pour d'autres tables que la table Vend car j'ai surtout créé le modèle MVC pour le site.
