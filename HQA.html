  • Harold's Quest - Aftermath est un "Visual Novel" codé en javascript et fonctionnant sur HTML. Il est la suite directe d'Harold's Quest et améliore le moteur de jeu existant.

    Le jeu dispose de nombreuses améliorations techniques qui lui permet d'être plus fluide et plus appréciable pour le joueur :



    - L'utilisation de HTML et de CSS couplé au Javascript permet une amélioration graphique conséquente, avec des personnages animés ou un effet d'écriture du texte.

    - Le jeu est disponible sur navigateur et sur téléphone car fonctionnant sur HTML. Toutes les dimensions d'écrans sont supportées, ce qui permet une grande flexibilité d'appareil.

    - Les données des tableaux sont stockées dans un fichier XML au lieu d'être raccroché au programme principal, ce qui permet de fluidifier la lecture du jeu par le navigateur.

    - Des combats réguliers ont été ajoutés, permettant de captiver le joueur dans son aventure. Ils demanderont de la réflexion pour pouvoir être fini.

    - Malgré l'absence d'objets à récupérer, le jeu arrive à compenser par une narration pleine de suspens qui pourra faire passer un bon moment chez nombre de joueurs.
